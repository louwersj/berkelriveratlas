# Media And Historic Maps

Media records are first-class atlas content objects.

- Historic photos, postcards, and maps can be described in Markdown.
- Rights metadata is required for published media records.
- Georeferenced map overlays can store bounds and default opacity.
- IIIF can be added later by extending the `files` and `rights` structures.

The sample repo uses local SVG placeholder assets to keep the public build open and reproducible.
