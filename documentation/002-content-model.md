# Content Model

The atlas uses one canonical Markdown file per content item.

- `object` records describe places, structures, river-related features, and other mappable entities.
- `media` records describe photos, maps, scans, and image-like objects with rights metadata.
- `story` records connect related objects through narrative themes.
- `page` records provide traditional static site content such as About and Methodology.

Each Markdown file contains front matter plus optional language sections for `en`, `de`, and `nl`.
