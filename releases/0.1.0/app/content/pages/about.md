## en

The Berkel River Atlas is a static, multilingual historical atlas of the Berkel river. This repository currently contains **sample data and implementation scaffolding** designed to be public, inspectable, and easy to grow.

## de

Der Berkel-Flussatlas ist ein statischer, mehrsprachiger historischer Atlas der Berkel. Dieses Repository enthält derzeit **Beispieldaten und Implementierungsgrundlagen**.

## nl
